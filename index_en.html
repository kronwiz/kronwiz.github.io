Work

My current work is primarily dedicated to the analysis necessary to define the technological architecture to be used in the projects, the workflows to be followed, and the procedures the development team must implement. I study new technologies to understand if they can be useful to improve the company products.

To a lesser extent than analysis work I do also the following work:

  • development of programs that process electronic documentation in XML and SGML and index it in the Solr search engine so that it can be searched by customers through search interfaces developed specifically. The main languages that I use are Python and C on Linux;
  • development of web applications whose server part is quite varied: there are CGIs written in C or Python, or applications written in Javascript and executed via node.js; the client part is built using standard techniques based on HTML, CSS, Javascript and Ajax requests to communicate with the server;
  • development of procedures to automate the management of the web farm through Bash scripts and containers managed by Docker; administration of Linux servers, managing the main services: SSH, FTP, NFS, etc.;
  • installation and configuration of server applications like Solr, Apache, MySQL; the web applications and the CMS that contains all the electronic documentation are based on these.

Besides using them on the workplace, the study of programming languages is a personal interest that I pursue also in my spare time. Because of this interest and the interest in teaching I started giving free training in some local CoderDojo groups.
