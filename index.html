Questa attività mi dà soddisfazione, ma scrivere programmi è più divertente, quindi conservo sempre alcuni lavori per me stesso, ottenendo così molteplici scopi: rendo più vario il lavoro, mi mantengo aggiornato e, infine, mi occupo di ricerca per migliorare le tecnologie adottate in azienda o per adottarne di nuove.

Quindi svolgo tutt'ora in prima persona i seguenti lavori:

  • sviluppo programmi che si occupano di elaborare la documentazione elettronica in formato XML e SGML e di indicizzarla nel motore di ricerca Solr in modo tale che sia reperibile dai clienti tramite interfacce di ricerca sviluppate appositamente. I principali linguaggi che uso sono Python e C in ambiente Linux;
  • sviluppo applicazioni web la cui parte server è piuttosto varia: ci sono CGI scritte in linguaggio C o in Python, oppure applicativi scritti in Javascript ed eseguiti tramite node.js; la parte client è realizzata usando le tecniche standard basate su HTML, CSS, Javascript e richieste Ajax per colloquiare con i server;
  • sviluppo procedure per automatizzare la gestione della sala macchine: script Bash, container Docker; amministro i server Linux, con la gestione dei servizi principali: SSH, FTP, NFS, ecc.;
  • installo e configuro applicazioni server come, per esempio: Solr, Apache, MySQL; questi ultimi sono alla base sia delle applicazioni web, sia del CMS che contiene tutta la documentazione elettronica.

Al di là del loro utilizzo in ambiente lavorativo, lo studio dei linguaggi di programmazione rimane comunque un interesse personale che perseguo anche nel tempo libero. Questa passione, unita a quella dell'insegnamento, mi ha portato a svolgere attività di formazione gratuita in alcuni gruppi di CoderDojo locali.
